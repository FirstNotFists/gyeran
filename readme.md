# 🥚 Egg Trigger (Prototype)

A tiny prototype library that displays an egg image on a specific screen area after user interaction — and makes it disappear shortly after.

This project is experimental and intentionally minimal.

---

## ✨ What This Library Does

1. Detects a user click on a specific area of the screen  
2. Displays an egg image after a configurable delay  
3. Removes the egg image automatically after a short duration  

That’s it. No state management, no heavy abstractions — just behavior.

---

## 🧠 Use Case Ideas

- Easter eggs / hidden UI interactions
- UI experiments or playful UX
- Interaction timing tests
- Prototyping delayed visual feedback

---

## 🚧 Project Status

This is a **prototype library**.

- APIs may change
- Features are intentionally limited
- No guarantees of backward compatibility (yet)
